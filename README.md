# Desafío Práctico 2 - Desarrollo de Servicios Web con Spring Boot

## Descripción del Proyecto
Este proyecto consiste en el desarrollo de un servicio web RESTful para la gestión de una institución educativa, desarrollado con 
**Java y Spring Boot**. El sistema permite la gestión de alumnos, profesores y materias.

## Características
* **Arquitectura:** Basada en capas (Controller, Service, Repository, Entity).
* **Servidor:** Despliegue automático en Apache Tomcat interno.
* **Base de Datos:** Persistencia en MySQL utilizando Spring Data JPA.
* **Documentación:** Implementación de Swagger/OpenAPI para pruebas de endpoints.

## Tecnologías Utilizadas
* **Lenguaje:** Java 17 (o la versión que estés usando)
* **Framework:** Spring Boot 3.x
* **Gestor de Dependencias:** Maven
* **Base de Datos:** MySQL
* **Documentación:** SpringDoc OpenAPI
